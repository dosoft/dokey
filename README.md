# dokey

Маленькое tray-приложение для Windows, позволяющее переключать раскладку клавиатуры клавишами **Shift**: левая — английская, правая — русская.

Основная идея заключается в том, что повторное нажатие одной и той же клавиши не меняет текущую раскладку. Это позволяет не помнить, какой у меня сейчас язык, просто нажимаешь нужную клавишу и уверен, что язык переключится.

Именно такой способ переключения я использовал ещё во времена DOS с **KeyRus**, а позже — в Windows с помощью **Punto Switcher**. Но Punto Switcher избыточно функционален и имеет ряд ограничений, поэтому в итоге я решил написать свою поделку.


## Установка

Скачайте `dokey-win-Setup.exe` со страницы [Releases](https://github.com/dosoft/dokey/releases) и запустите его. Приложение установится автоматически и добавится в автозагрузку.

**Примечание:** приложение и установщик пока не подписаны цифровой подписью, поэтому Windows (SmartScreen / Защитник Windows) может показать предупреждение. Это ожидаемо — нажмите «Подробнее» → «Выполнить в любом случае», чтобы продолжить установку.

## Использование

- Нажмите **правый Shift** — русская раскладка.
- Нажмите **левый Shift** — английская раскладка.
- Иконка в трее показывает текущую раскладку активного окна.
- Двойной клик по иконке открывает настройки.

## Условие срабатывания

Переключение происходит только если Shift нажат **чисто**:
- без одновременных клавиш (Ctrl, Alt, Win, другие буквы/стрелки);
- без клика мышью во время удержания;
- Если вы нажали и держили шифт больше 300 мс (настраивается), то язык не переключается.

Обычное поведение Shift (Shift+буква, Shift+стрелка, Shift+click) не нарушается.

## Требования

- Windows 10 / 11.
- Установленные раскладки RU (`0419`) и EN (`0409`) в языковых настройках Windows.
- Установщик уже включает .NET Runtime (self-contained сборка).

## Обновления

Приложение использует [Velopack](https://github.com/velopack/velopack). Обновления проверяются автоматически; при появлении новой версии появится уведомление в трее.

## Известные ограничения

- **Окна с повышенными правами (UAC):** если активное окно запущено от администратора, а dokey — нет, переключение в нём не работает. Решение: запустить `dokey.exe` тоже от администратора. Встроенный автозапуск так и делает.
- Поддерживаются только две раскладки: RU (`0x0419`) и EN (`0x0409`).
