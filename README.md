Небольшое приложение для Windows, которое переключает раскладку клавишами **Shift**. По умолчанию левая клавиша включает английский язык, а правая — русский; это назначение можно поменять в настройках.

Главная особенность в том, что повторное нажатие той же клавиши не меняет раскладку. Не нужно помнить, какой язык сейчас выбран: достаточно нажать нужный **Shift** и продолжить печатать.

Такой способ переключения я использовал ещё во времена DOS с **KeyRus**, а позже — в Windows с помощью **Punto Switcher**. Но Punto Switcher оказался для этой задачи слишком громоздким и не всегда удобным, поэтому я решил написать своё небольшое приложение.

## Установка

Скачайте `dokey-win-Setup.exe` со страницы [Releases](https://github.com/dosoft/dokey/releases) и запустите его. Приложение установится автоматически. При первом запуске dokey добавляется в автозагрузку (может появиться запрос UAC), чтобы запускаться вместе с Windows. Отключить автозапуск можно в настройках приложения.

**Примечание:** приложение и установщик пока не подписаны цифровой подписью, поэтому Windows (SmartScreen / Защитник Windows) может показать предупреждение. Это ожидаемо — нажмите «Подробнее» → «Выполнить в любом случае», чтобы продолжить установку.

## Использование

- По умолчанию нажмите **правый Shift** для русской раскладки.
- По умолчанию нажмите **левый Shift** для английской раскладки.
- При необходимости поменяйте назначение левой и правой клавиш в настройках.
- Иконка в трее показывает текущую раскладку активного окна.
- Двойной клик по иконке открывает настройки.

## Условие срабатывания

Переключение происходит только если Shift нажат **чисто**:

- без одновременных клавиш (Ctrl, Alt, Win, другие буквы/стрелки);
- без клика мышью во время удержания;
- если вы удерживали Shift дольше 300 мс (настраивается), язык не переключается.

Обычное поведение Shift (Shift+буква, Shift+стрелка, Shift+click) не нарушается.

## Требования

- Windows 10 / 11.
- Установленные раскладки RU (`0419`) и EN (`0409`) в языковых настройках Windows.
- Установщик уже включает .NET Runtime (self-contained сборка).

## Обновления

Приложение использует [Velopack](https://github.com/velopack/velopack). Обновления проверяются автоматически; при появлении новой версии появится уведомление в трее.

## Известные ограничения

- **Окна с повышенными правами (UAC):** если активное окно запущено от администратора, а dokey — нет, переключение в нём не работает. Решение: запустить `dokey.exe` тоже от администратора. Встроенный автозапуск так и делает.
- Поддерживаются только две раскладки: RU (`0x0419`) и EN (`0x0409`).
