Download for WindowsView on GitHub

Why DOKEY?

Standard Windows keyboard-layout shortcuts have a drawback: you need to remember which language is active and therefore whether you need to switch at all. Assigning layouts to Shift solves this: Left Shift for English and Right Shift for Russian. You can even press the same key again; the layout will not change. You can swap the assignments in Settings.

I first used this switching method in DOS with KeyRus, and later in Windows with Punto Switcher. Punto Switcher eventually became too heavyweight and inconvenient for this single task, so I created a small dedicated application.

Features

Direct switching

One key selects the exact layout instead of cycling through a list.

Normal Shift behavior

Shift combined with letters, arrows, mouse clicks or modifier keys works as usual.

System tray

The tray icon shows the active window's current EN or RU layout.

Customizable

Swap left/right assignments and adjust the maximum press duration.

Automatic updates

Checks GitHub Releases and notifies you when a new version is available.

Private and free

No telemetry or analytics. Open source under the MIT License.

How to install

  1. Download dokey-win-Setup.exe from Releases.
  2. Run the per-user installer; no separate .NET installation is required.
  3. DOKEY starts in the tray. Open Settings to change key assignments or auto-start.

Windows SmartScreen may warn that the installer is unsigned. Choose “More info” and “Run anyway” if you trust the download.
